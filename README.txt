8) ¿Qué es el IL?

El lenguaje intermedio es lo que se genera al compilar codigo de .net, lo que seria codigo administrado. 
Ese tipo de codigo es el que se ecnuentra entre el compilador (codigo en visual) y el codigo de maquina 101010

9) Explique con sus palabras VES, CTS y CLS.

VES: Es el ambiente que se encarga de compilar y correr aplicaciones que fueron hechas en .net
CTS: Es el tipo de tipeado que tiene que seguir el codigo en c# , para ser compatible y compilado en el IL
CLS: Es el tipo de lenguaje que tiene que seguir el codigo en c# , por ejemplo al usar visual studio o visual basic

10) Explique con sus palabras que es un Assembly, el early y late Binding.

ASSEMBLY:_ es el lenguaje que usan los microprocesadores
ERLY BINDING: es cuando usamos un objeto declarando el tipo y el compilador ya sabe leer sus caracteristicas
LATE BINDING: es cuando pasamos de forma abstracta un objeto donde el compilador no sabe nada de ese objeto
